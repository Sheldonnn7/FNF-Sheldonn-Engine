# Friday Night Funkin' - Sheldonn Engine

An engine built off of Psych Engine 0.6.3, because idk.

## Installation:

1. Install Haxe 4.2.5, not the latest release. 
https://haxe.org/download/version/4.2.5


2. Install Git.
https://git-scm.com/


3. Install Visual Studio 2019: https://visualstudio.microsoft.com/vs/older-downloads/


4. Run haxelib install hmm


5. Then run hmm install


6. Run "lime test windows" into the command prompt. (or go in the art folder and use the build .bat files.)
